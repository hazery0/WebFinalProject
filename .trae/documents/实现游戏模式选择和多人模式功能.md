# 实现游戏模式选择和多人模式功能

## 1. 前端页面创建

### 1.1 游戏模式选择页面

* 创建 `GameModeSelect.vue` 组件

* 包含两个按钮：单人模式和多人模式

* 单人模式跳转至现有游戏页面

* 多人模式跳转至多人模式页面

### 1.2 多人模式页面

* 创建 `MultiplayerMode.vue` 组件

* 包含两个按钮：创建房间和加入房间

* 加入房间时需要输入房间号

### 1.3 多人游戏房间页面

* 创建 `MultiplayerGame.vue` 组件

* 基于现有 `HistoricalFigureGame.vue` 改造

* 替换标题为当前房间的玩家ID

* 添加房间信息显示

* 实现基本的房间内通信

## 2. 路由配置

### 2.1 更新路由配置

* 添加 `/mode-select` 路由

* 添加 `/multiplayer` 路由

* 添加 `/multiplayer-room/:roomId` 路由

* 更新 `/start` 跳转逻辑，跳转到模式选择页面

### 2.2 路由导航

* 游戏开始页面 → 模式选择页面

* 模式选择页面 → 单人游戏页面 或 多人模式页面

* 多人模式页面 → 创建房间 或 加入房间

* 创建/加入房间 → 多人游戏房间页面

## 3. 状态管理

### 3.1 房间管理

* 实现本地房间创建和加入逻辑

* 生成随机房间号

* 模拟房间内玩家ID分配

### 3.2 游戏状态

* 共享游戏状态（目标人物）

* 同步玩家猜测

* 显示玩家列表

## 4. 功能实现

### 4.1 游戏模式选择

* 页面设计：简洁的选择界面

* 交互逻辑：按钮点击跳转

### 4.2 多人模式

* 创建房间：生成随机6位房间号

* 加入房间：输入房间号验证

* 房间列表：显示当前房间信息

### 4.3 多人游戏房间

* 标题显示：当前房间号和玩家ID

* 游戏逻辑：与单人模式相似，但共享目标人物

* 玩家交互：显示其他玩家的猜测
