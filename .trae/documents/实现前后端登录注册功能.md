# 实现前后端登录注册功能

## 后端实现步骤

1. **添加依赖**
   - 添加 JWT 相关依赖
   - 添加密码加密依赖

2. **配置数据库**
   - 修改 `application.properties` 配置 MySQL 连接
   - 配置 JPA 自动建表

3. **创建 User 实体类**
   - 包含字段：id、username、password、isAdmin
   - 配置 JPA 注解

4. **创建 UserRepository**
   - 继承 JpaRepository
   - 添加根据用户名查询用户的方法

5. **创建工具类**
   - JWT 工具类：生成和验证令牌
   - 密码加密工具类：使用 BCrypt 加密密码

6. **创建 AuthController**
   - 实现注册接口：接收用户名和密码，加密后保存到数据库，isAdmin 默认为 0
   - 实现登录接口：验证用户名和密码，生成 JWT 令牌返回

7. **配置 CORS**
   - 允许前端跨域请求

## 前端实现步骤

1. **添加依赖**
   - 添加 axios 用于 HTTP 请求

2. **创建组件**
   - LoginView.vue：登录页面
   - RegisterView.vue：注册页面

3. **配置路由**
   - 添加登录和注册路由

4. **创建 API 服务**
   - 封装 axios 请求
   - 处理 JWT 令牌的存储和携带

5. **实现登录注册逻辑**
   - 登录：发送用户名密码，保存返回的 JWT 令牌
   - 注册：发送用户名密码，注册成功后跳转到登录页面

6. **创建状态管理**
   - 使用 Pinia 管理用户登录状态

## 技术栈

- 后端：Spring Boot 3 + JPA + MySQL + JWT
- 前端：Vue 3 + TypeScript + Vue Router + Pinia + Axios

## 预期效果

- 前端可以访问登录和注册页面
- 注册功能：用户可以输入用户名和密码注册，密码加密存储
- 登录功能：验证用户名密码，返回 JWT 令牌
- 前后端成功联调，实现完整的认证流程
